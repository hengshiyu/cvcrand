# cvcrand 0.0.3

## Major changes

1. Added the function of `cvrcov()` for covariate-by-covariate constrained randomization. 
    
2. changed the function name `cvcrand()` to be `cvrall()` for constrained randomization. 

3. Added the arguments `check_validity` to check validity by giving outputs of cluster pair coincidence matrix and descriptive statistics.
